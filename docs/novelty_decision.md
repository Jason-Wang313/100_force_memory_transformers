# Novelty Decision

Decision: conditionally promising.

The v5-expanded evidence supports keeping the idea alive as STRONG_REVISE. The mechanism is more specific than generic transformer memory or generic uncertainty filtering: it targets persistent hidden force state in contact-rich manipulation and couples force-event tokens, decay, reset, diagnostic probes, calibrated risk, and tail-risk planning.

The novelty is not yet enough for ICLR main because trained-model evidence, external benchmarks, hardware or accepted high-fidelity evidence, and manual prior-work depth are missing.
